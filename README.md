# 🎯 Professional AI Interview Analyzer

An AI-powered interview analysis platform built using Python and Streamlit that evaluates interview recordings and provides professional insights on communication quality, confidence, sentiment, and filler word usage.

---

# 🚀 Features

✅ Audio Upload Support  
✅ Speech Transcript Analysis  
✅ Filler Word Detection  
✅ Communication Scoring  
✅ Sentiment Analysis  
✅ Interactive Dashboard  
✅ Professional Radar Charts  
✅ Overall Performance Score  
✅ Streamlit Web Interface  

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/charanakula07-ui/Professional-AI-Interview-Analyzer.git
```

## 2️⃣ Open Project Folder

```bash
cd Professional-AI-Interview-Analyzer
```

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

## 4️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## 5️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
streamlit run app.py
```
